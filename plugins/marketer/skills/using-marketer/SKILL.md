description: |
  Use when any marketing, brand, positioning, naming, go-to-market, social
  writing (X/Twitter, LinkedIn), AI-slop removal, or presentation work comes
  up — routes substantial work to the marketer agent and orients quick
  lookups. Advisory, not mandatory.
---

For strategy, positioning, multi-step content, or anything that spans several
marketing concerns: dispatch the `marketer` agent — it carries skill-index
discovery, project brand/GTM context, and channel best practices, and pulls in
the right `marketer:*` skills itself.

For a quick single-purpose lookup, invoke the matching `marketer:*` skill
directly instead of spawning the agent (for example `marketer:naming` for a
name, `marketer:x-writing` for one post).

The agent path is the default for real work; the direct-skill path is the
shortcut for a one-off. This skill lists no skills of its own — the marketer
agent and the skill index own that enumeration so it never goes stale here.
