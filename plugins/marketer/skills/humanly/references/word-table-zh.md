# 中文 AI 廢話替換表（詞彙與句型）

> **角色**：中文禁用詞彙與禁用句型的唯一來源（source of truth）。
> **新增或修改詞彙、句型 → 只改這個檔**（先判斷 Tier），改完跑 `python3 scripts/build-prewrite.py` 重新生成 prewrite 檔。
> **誰讀我**：rewrite / review mode 直接讀本檔全文；prewrite mode 讀 `generated/prewrite-zh.md`（由腳本自動抽取本檔的 Tier 1 與禁用句型段，不要手動複製到別處）。

三級分類，依 AI 生成文本中的出現頻率判斷。

- **Tier 1 — 必換。** AI 極度愛用，看到就換。
- **Tier 2 — 聚集時才換。** 單獨出現 OK，同段落出現 2 個以上就是 AI 味。
- **Tier 3 — 高密度才換。** 正常詞彙，但 AI 用太多。整篇充斥時才處理。

## Tier 1 — 必換

| 替換 | 用 |
|---|---|
| 賦能 | 幫助、協助 |
| 打造 | 做、建立、開發 |
| 全方位 | 完整、各方面 |
| 落地 | 實行、執行、做出來 |
| 底層邏輯 | 基本原理、根本原因 |
| 顆粒度 | 細節程度、精細度 |
| 抓手 | 方法、著力點、切入點 |
| 拉齊 | 統一、對齊、同步 |
| 鏈路 | 流程、路徑、環節 |
| 護城河 | 優勢、門檻、競爭力 |
| 賽道 | 領域、方向、市場 |
| 深度（空洞修飾）| （刪除，或具體描述深在哪）|
| 深入探討 | 看看、來看、聊聊 |
| 此外 / 另外值得一提的是 | （刪除，直接說下一件事）|
| 至關重要 / 極其重要 | 重要、關鍵 |
| 不可磨滅 | 深刻、持久 |
| 作為...的證明 / 體現 | 說明了、顯示 |
| 標誌著 | （直接描述發生了什麼）|
| 見證了 | 看到了、經歷了 |
| 凸顯 / 彰顯 | 顯示、反映 |
| 為...奠定基礎 | 幫助、促成 |
| 充滿活力 | （具體描述活力在哪，或刪除）|
| 不斷演變（的格局）| 在變（的環境）|
| 深深植根於 | 來自、源於 |
| 硬撐 / 在撐 | 撐著、忍著、勉強做（或具體描述狀態）|
| 踩坑 | 遇到問題、碰到雷 |
| 爆了 | 大量、暴增、很多人（依語境具體描述）|
| 很真實 / 是有效的（修飾情緒）| 很正常、很合理（手法見 patterns #38）|
| 不該被指責 / 這不是你的錯 | （刪除，或「不怪你」）|
| 值得被看見 / 你的感受很重要 | （刪除，或講具體看到什麼）|

## 禁用句型 — 看到就刪

宣告式、鋪陳式的句型，等同 Tier 1（完整脈絡與改寫手法見 patterns #22）：

| 類型 | 禁用範例 |
|------|----------|
| 假裝驚訝 | 「說出來你可能不信」「你一定會很驚訝」 |
| 過度強調 | 「這件事非常非常重要」「絕對不能錯過」「一定要看到最後」 |
| 空洞鋪陳 | 「我發現一件事」「讓我告訴你一個秘密」「先說重點」 |
| 冗餘確認 | 「是的，你沒看錯」「沒錯，就是這樣」 |
| 誇張情緒 | 「太棒了！」「天啊！」「超級重要！」「真的太厲害！」 |
| 數據宣告 | 「結果很驚人：」「這份數據顯示：」 |
| 回音式開場 | 「『X』這句話超有感」「你說得沒錯」「X 真的是」「超多人也都走過這段」 |
| 空洞肯定 | 「這種進度很實在」「這個很不錯」 |
| 假坦白鉤子 | 「說真的」「老實說」「講白了」「跟你說個秘密」當開場（patterns #44） |
| 金句公式 | 「注意力是新的貨幣」「真正的 X，是 Y」「拖延是完美主義的影子」（patterns #43） |
| 勸誡反問 | 「與其……不如……，不是嗎？」「機會永遠留給準備好的人」（patterns #46） |

禁止純粹為了開場而存在的肯定句，去掉那句不影響整段意思，就是填充物。社群回覆時，禁止先複述對方內容再表示同意（「你說X → 我覺得X」），直接從自己的視角切入。

## Tier 2 — 同段落出現 2+ 才換

### 做作動詞（比喻性動詞包裝）

人在做事，但動詞選得太文青、太有設計感。跟朋友聊天不會這樣講就換掉。

| 替換 | 用 |
|---|---|
| 接不住 | 搞不來、做不了 |
| 撐不起來 | 做不到 |
| 扛不動 | 處理不了 |
| 餵給它 | 給它、丟進去 |
| 承接 | 做、處理 |

### 一般 Tier 2

| 替換 | 用 |
|---|---|
| 沉澱 | 累積、整理 |
| 迭代 | 改進、調整、改版 |
| 閉環 | 完整流程、從頭做到尾 |
| 心智 | 認知、印象、想法 |
| 觸達 | 接觸到、送到 |
| 降本增效 | 省錢提效、降低成本 |
| 生態 / 生態系 | 系統、環境、圈子 |
| 矩陣 | 分類表、組合 |
| 增強 | 加強、改善 |
| 培養 / 促進 | 幫助、推動 |
| 強調（動詞）| （刪除，直接說重點）|
| 展現 / 展示（空泛動詞）| 給具體例子，或刪除 |
| 寶貴的 | 有用的、有價值的 |
| 格局（抽象名詞）| 環境、情況、局面 |

## Tier 3 — 高密度才換

| 詞 | 怎麼處理 |
|---|---|
| 重要的 / 非常重要 | 換成具體說明為什麼重要 |
| 顯著的 / 顯著地 | 給數字或比較 |
| 創新 / 創新性 | 描述新在哪裡 |
| 有效的 / 有效地 | 說怎麼有效，或引用數據 |
| 關鍵 / 關鍵性的 | 說明為什麼是關鍵 |
| 獨特的 | 描述獨特在哪 |
| 令人印象深刻 | 說什麼讓人印象深刻 |
| 值得注意的是 | （刪除，直接說）|
| 突出的 / 突出地 | 給具體事例 |
| 豐富的（比喻）| （具體描述豐富在哪）|
