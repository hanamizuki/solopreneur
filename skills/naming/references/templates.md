# Naming Skill — Document Templates

Each phase writes a Markdown file to `{repo}/docs/naming/` following the
template below. Templates exist so that `resume` mode can reliably re-read
state, and so that different agents / sessions produce consistent output.

## test-kit.md

```markdown
# Tension Test Kit

> Generated by naming skill · {date}
> Instructions for the user to run with 3-10 testers

## Who to invite

- **5-10 testers ideal. 3 is the minimum.**
- Mix of ideal customers (60%) and trusted advisors (40%).
- **Exclude:** co-founders, team members, anyone with internal context.
- Individual 1-on-1s, not focus groups (focus groups push toward consensus = invisible zone).

## Protocol (per tester)

1. Frame: "I'm testing name candidates for a new [product / company]. I'll show you each name cold — no logo, no pitch. I want your first reactions."
2. For each of the 10 names:
   - Say the name out loud.
   - Write it down / show it on screen.
   - Wait 5 seconds silently.
   - Ask the three questions below (in order).
   - Record verbatim if possible.
3. Do NOT explain what the product does until AFTER all 10 names are done.

## Questions (same three, every name)

1. "What does this name make you feel?" (NOT "do you like it")
2. "What kind of company or product do you imagine?"
3. "If you saw this on a billboard, would you remember it tomorrow?"

## Recording Table

Copy this table once per tester:

**Tester: {name or initials} · Role: {ICP / advisor} · Date: {date}**

| # | Name | Feeling (verbatim) | Imagined category | Remember tomorrow? (Y/N) |
|---|------|-------------------|-------------------|-------------------------|
| 1 | {name} | | | |
| 2 | {name} | | | |
| ... | ... | | | |
| 10 | {name} | | | |

## When results are in

Return to this conversation and paste the filled tables. The skill will
interpret them using the rules in SKILL.md Phase 5.

## Day-after memorability check

If possible, message each tester ~24 hours later with:
> "Quick one — of the names I showed you yesterday, which do you remember?"

Record which names are recalled unprompted. This is the strongest signal
in the whole test kit.
```
